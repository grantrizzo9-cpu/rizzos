# **App Name**: Affiliate AI Host

## Core Features:

- NVMe Edge Hosting: Provide high-performance NVMe web hosting with fast loading speeds.
- AI Content Studio: Offer tools for generating SEO-optimized blogs, marketing ads, videos, and complete niche sites powered by Gemini 2.5 Flash using Genkit tool.
- Affiliate Network: Allow affiliates to earn 70-75% recurring daily commissions in AUD.
- Real-Time Dashboard: Display real-time AUD earnings, active referrals, and commission rate in the user dashboard.
- Strategy Center: Provide educational articles (6 articles, ~500 words each) to help affiliates scale their referrals.
- Payout Settings: Allow users to input their PayPal email for automated 24-hour commission payouts.
- User Authentication: User Authentication Using Firebase Auth
- Hero Image: Use the 'HOSTPROAI' image as the hero image on the landing page.

## Style Guidelines:

- Primary color: Professional blue (#3b82f6) to convey trust and reliability.
- Accent color: Urgency red (#ef4444) for highlighting important alerts and CTAs.
- Background color: Slate gray (#26272b), desaturated to 20%, for a modern 'command center' feel.
- Headline font: 'Space Grotesk' sans-serif for headlines to create a modern, tech-focused look.
- Body font: 'Inter' sans-serif for body text to ensure readability and clarity.
- Use sharp, minimalist icons to represent features and data points, fitting with the 'command center' design language.
- Information-dense pages to establish technical authority and a sense of high-performance.
